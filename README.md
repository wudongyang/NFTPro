# NFTPro
汇聚主流 NFT 交易平台，提供专业交易功能。

# 专业功能

1. 聚合主流 NFT 交易平台
2. 批量挂单
3. 批量阶梯挂单
4. 根据 Rank 批量挂单
5. 批量调价
6. 低价报警

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
